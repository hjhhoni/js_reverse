Promise 中的 resolve 和 reject 参数

基本概念
    resolve 和 reject 是 Promise 构造函数中的两个回调函数参数
    它们用于控制 Promise 实例的状态转换和结果传递

主要功能
    1. 状态转换
        resolve: 将 Promise 状态从 pending（等待）转换为 fulfilled（已完成）
        reject: 将 Promise 状态从 pending（等待）转换为 rejected（已拒绝）
    2. 结果传递
        resolve(value): 传递成功的值，该值会被 .then() 方法接收
        reject(reason): 传递失败的原因，该原因会被 .catch() 方法接收


new Promise(function (resolve, reject) {
    // 异步操作完成后
    if (/* 成功条件 */) {
        resolve('成功数据');  // 触发 fulfilled 状态
    } else {
        reject('失败原因');    // 触发 rejected 状态
    }
})


状态转换流程
    Promise 初始状态：pending
    调用 resolve → 状态变为：fulfilled
    调用 reject → 状态变为：rejected
    状态一旦改变后不可逆转

与后续处理的关系
    resolve 的参数 → 传递给 .then() 的第一个回调函数
    reject 的参数 → 传递给 .catch() 的回调函数
