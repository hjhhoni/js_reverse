在python中有虚拟环境的概念, 创建虚拟环境大多使用conda中的 conda create -n 创建
nvm其实是nodejs中的解释器管理工具, 我们可以利用nvm创建不同的node执行环境

nvm的使用方式:
    1.nvm list: 列出当前机器上可用的所有node环境
    2.nvm install <版本号>: 下载指定版本的node解释器(16版本、18版本、22版本)
    3.nvm use <版本号>: 可以基于你写的指令完成node解释器的版本切换

本次js逆向课程的主解释器版本为18版本, 22版本主要是用于驱动AI的, 16版本主要是解决抖音逆向的
